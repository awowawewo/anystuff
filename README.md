Description
Pearl Abyss used a custom font for Korean that matches the game's aesthetic. For all other languages, including Russian, a different font was used. To fix this, I manually drew all the missing characters based on the existing English set, resulting in this font.

Installation Instructions
Install via JSON Mod Manager: Drag and drop the archive onto the "Drop mods here to install" area, enable the mod using the toggle switch in the list, and click "Apply Mods".

Описание
Pearl Abyss использовали для корейского языка особый шрифт, подходящий эстетике игры. Для всех остальных языков, в том числе русского, они использовали другой шрифт. Поэтому я вручную отрисовал все недостающие буквы на основе английских (которые уже были) , и в итоге получился этот шрифт.

Инструкция по установке
Установка через JSON Mod Manager - перекиньте архив на Drop mods here to install, включите мод в списке тумблером и нажмите Apply Mods.
